Exercise 2.1
Scenario: You see data things

Objective:

Learn about commonly used data types in JavaScript: boolean, integer, float (floating-point number, aka double) string. These are also known as primitives.
Steps:

Type the following thing into console:

true
This is a boolean. There can be only 2 possible values: true or false

false
You can read up more about Booleans.

Here's an integer (numbers):

9999
The largest number value is approximately 1.79E+308, or 21024.

For bigger numbers, we use BigInt.

You can see the type of data by typing:

typeof 9999
Here's a string:

"Hello world"
You can manipulate strings:

"tech ladies".toUpperCase()

"la la. ".repeat(10)

'The quick brown fox jumps over the lazy dog. If the dog reacted, was it really lazy?'.replace('dog', 'monkey')
You can read up more about Strings.

Here's a floating number:

3.146
When you are unsure:

null
